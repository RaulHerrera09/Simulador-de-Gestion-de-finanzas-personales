# Características  
 Registro de ingresos y gastos  
 Cálculo automático del saldo

 Gráfico interactivo con *Chart.js*  
 Diseño moderno con *Bootstrap 5*  

# Tecnologías Usadas  
- HTML5, CSS3, JavaScript  
- Bootstrap 5  
- Chart.js  
- LocalStorage
